# The Open Syllabus

<p align="center"><strong>An open-source AI-powered academic platform for students to organize, discover, and collaborate around educational resources.</strong></p>

---

## Overview
The Open Syllabus is a modern learning platform built with React, Vite, Firebase and Express. It centralizes syllabi, notes, discussions and AI-assisted learning into a single experience while remaining fast, responsive and easy to contribute to.

## Highlights
- Modern responsive interface
- Firebase Authentication & Firestore
- AI study assistant (Gemini API)
- Course resources and discussions
- Dark mode and study mode
- Production-ready architecture

## Technology Stack
| Layer | Technology |
|---|---|
| Frontend | React 19, Vite |
| Styling | Tailwind CSS |
| Backend | Express |
| Database | Firebase Firestore |
| Authentication | Firebase Auth |
| AI | Gemini API (optional) |

## Quick Start
```bash
git clone https://github.com/Saksham1105/The-Open-Syllabus.git
cd The-Open-Syllabus
npm install
npm run dev
```

## Environment Variables
Create `.env.local`:
```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```
